0. Общие моменты
Приложение представляет собой симулятор экзамена
В нём можно провести экзамен, получив после прохождения подробный отчёт о количестве набранных баллов и об ошибках
Приложение поддерживает добавление вопросов через форму, а так же их удаление

1. Прохождение экзамена
Нажать кнопку "Начать экзамен"
Перелистывать вопросы можно, нажимая кнопки "Назад" - "Вперёд"
После завершения ответов на вопросы нажать "Завершить экзамен"
Если на какие-то вопросы не было дано ответа, то появится предупреждение с номерами тех вопросов, на которые не был дан ответ
В случае, если таковых нет или пользователь решил принудительно завершить экзамен, не дав на некоторые вопросы ответ, то приложение
перейдёт к форме результатов экзамена
В форме результатов экзамена отображается:
	- Результат: сдал/не сдал
	- Количество правильных ответов
	- Процент выполнения экзамена
	- Отчёт о данных ответах
	- Правильные ответы
Отчёт можно сохранить в файл, предварительно введя имя в соотвествующее поле

2. Добавление вопросов
Выбрать тип вводимого вопроса
Выбрать количество вариантов ответа, если выбран тип вопросов с выбором
Нажать "Сгенерировать форму"
Внизу появится форма для добавления вопроса
В ней ввести формулировку и ответ:
	- Для текстового вопроса - строку в текстовое поле
	- Для вопроса с одним верным ответом - варианты ответов в текстовые поля (также отметить
	  верный ответ напротив соотвествующего текстового поля)
	- Для вопроса с несколькими верными ответами - варианты ответов в текстовые поля (также отметить
	  верные ответы напротив соотвествующего текстового поля)
Нажать добавить
Для ввода нового вопроса форма очистится автоматически, если предыдущий вопрос был введён корректно

3. Автоматическое удаление/блокирование(деблокирование) вопросов
Чтобы удалить вопрос, нужно нажать кнопку "Режим удаления вопросов"
Выбрать в выпадающем списке нужный вопрос и нажать "Удалить"

Блокирование(деблокирование) вопросов выполняется аналогично

4. Ручное добавление/удаление/блокирование вопросов
Текстовые файлы с вопросами располагаются в папке "exam\bin\Debug\Questions\"
Шаблон вопроса выглядит следующим образом:
	- Для текстовго вопроса : "Вопрос | Ответ"
          Пример: Размер кадра Ethernet ( в байтах )? | 1526
	  Вопросы этого типа располагаются в файле textQuestion.txt
	- Для вопроса с одним верным ответом: "Вопрос | Ответ 1 | Ответ 2 | Ответ 3 ..."
          Верный ответ помечается знаком ! перез знаком | для данного варианта
          Пример: По какому порту работает Telnet? | 21 | 22 |! 23 | 24 
	  Вопросы этого типа располагаются в файле oneChoiceQuestion.txt
	- Для вопроса с несколькими верными ответами: "Вопрос | Ответ 1 | Ответ 2 | Ответ 3 ..."
          Верный ответ помечается знаком ! перез знаком | для данного варианта
          Вопрос? |! Правильный ответ 1 |! Правильный ответ 2 | Неправильный ответ 1 | Неправильный ответ 2 
	  Вопросы этого типа располагаются в файле multiplyChoiceQuestion.txt

Удаление и добавление вопросов осуществляется удалением/добавлением в файл соответствующих строк

Также можно временно заблокировать вопрос, чтобы он не появился во время экзамена
Сделать это можно, поставив в начале строки вопроса знак #

Также можно изменить время проведения экзамена в файле time.txt
	- Первая строка - часы
	- Вторая строка - минуты
